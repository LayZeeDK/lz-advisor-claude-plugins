The script:
1. Splits eval queries 60/40 train/test (stratified by should_trigger)
2. Evaluates each query 3 times via claude -p with --model claude-sonnet-4-6
3. Uses extended thinking (anthropic SDK) to propose description improvements
4. Iterates up to 5 times or until all train queries pass
5. Returns best_description selected by TEST score (avoids overfitting)

Output: results directory with iteration-N/ subdirectories, final results.json with best_description.
