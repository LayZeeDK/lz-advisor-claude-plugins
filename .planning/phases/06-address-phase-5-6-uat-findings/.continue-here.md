Use "Use this tool when..." (calm imperatives) and reserve all-caps imperatives for compliance rules only. Do NOT add MUST / REQUIRED / CRITICAL to orient prescriptions to "force" tool use.
- [ ] CONSTRAINT: Personal-tool-name references in marketplace plugin SKILL body — the lz-advisor plugin is published as a universal marketplace plugin. Any references to user-specific tools (e.g., `markdown.new`, `url-to-markdown`, `playwright-cli`, `[username]'s CLAUDE.md`) couple the plugin to a single user's environment. Do NOT add personal-tool examples to plugin SKILL.md or references/.

Stage 1 KCB FAILing on plugin 0.8.x does NOT mean Pattern D fails on real prompts. Always validate empirical gate verdicts against realistic UAT prompts before declaring Phase failures.
