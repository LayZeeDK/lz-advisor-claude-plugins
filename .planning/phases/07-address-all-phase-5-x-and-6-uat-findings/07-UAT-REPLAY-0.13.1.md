  2. **residual-shape-regression-parser** (new from this UAT): Smoke fixture parser regex `/^[^:\s]+:\d+...` rejects backtick-wrapped fragment-grammar lines that are otherwise valid. Two mitigation paths:
     - (a) Loosen parser to accept optional leading backtick: `/^\`?[^:\s]+:\d+...\`?/` (parser-side fix; smoke gate becomes more permissive)
     - (b) Tighten agent prompt to forbid backtick wrapping (model-side fix; smoke gate stays strict)
     Per "agents shouldn't know about smoke gate tolerance" directive 2026-05-07, Path (a) is preferred.

  3. **residual-pfv-outlier-cap** (new from this UAT + n=10 evidence): PFV per-entry +10% outlier cap (66w) is too tight for security-reviewer's auto-clarity carve-out (CVE-class findings need full prose).
