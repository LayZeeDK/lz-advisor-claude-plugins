## 6-Session UAT Pipeline (D-03 Stage 2)

RESEARCH.md recommends running the pipeline as **six separate `claude -p` invocations** with artifact handoff via git worktree state (not one integrated session). Pipeline shape:

1. `/lz-advisor.plan <task>` -> writes `plans/<slug>.plan.md`
2. `/lz-advisor.execute @plan` -> commits code changes
3. `/lz-advisor.review` (NO plan attached; Finding J re-exercise) -> review findings
4. `/lz-advisor.plan "address review findings"` -> writes `plans/address-review-findings.plan.md`
5. `/lz-advisor.execute @address-review-findings.plan.md` -> commits more changes
6. `/lz-advisor.security-review` -> security findings

**Orchestration ownership (committed):**

- **Execution owner:** `/gsd-verify-work` at phase close is the committed execution owner for the 6-session UAT pipeline. This closes D-03 Stage 2 at the phase boundary.
- **Harness stub path (TBD work):** `.planning/phases/05.4-address-uat-findings-a-k/smoke-tests/uat-pipeline/runner.mjs` — a Node.js ESM script that sequences the six `claude -p` invocations with git-worktree state handoff between sessions. The script does not yet exist; creating it is TBD work explicitly delegated to `/gsd-verify-work` (or a follow-up quick task if the verify step elects to implement the harness itself). Phase-close is not blocked on pre-creating the script; it may be authored inside the verify step.
- **Acceptance gate at phase close (D-04-equivalent, inherited from Phase 5.3):** >=80% first-try advisor success AND <=4 tool-calls-per-consultation average.

**Emergency fallback (NOT a valid passing state):** If the 6-session harness cannot be completed at phase close (e.g., a Claude Code CLI bug blocks multi-session worktree orchestration), document the blocker in `05.4-UAT-RESULTS.md` and open a follow-up quick task to land the full pipeline. A 3-session variant (plan -> execute -> review) may be run for partial signal only; it does NOT satisfy the Stage 2 acceptance gate on its own. The pipeline must reach 6 sessions before the phase is considered closed.
