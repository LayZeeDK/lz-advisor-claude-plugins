Strip the `(formerly High)` / `(formerly Medium)` residue (AGNT-01, no third-generation residue). Retarget the security budget fixture's parser in lockstep, keeping the `[Axx]` assertion and the 75w CVE auto-clarity carve-out.

Purpose: The second half of the atomic few-shot-drift guard (D-11). The security-reviewer carries the higher-stakes contract -- OWASP provenance and the CVE carve-out are security controls (RESEARCH Security Domain); a partial rewrite that drops `[Axx]` or the CVE example degrades the security finding's provenance.

Output: A security-reviewer agent emitting the unified grouped grammar with OWASP tags preserved, gated by its retargeted fixture.
</objective>
