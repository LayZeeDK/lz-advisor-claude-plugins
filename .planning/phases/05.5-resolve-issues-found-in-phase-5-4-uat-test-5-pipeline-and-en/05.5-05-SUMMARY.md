## Drift correction note

**Pre-existing Phase 5.4 drift context (for future readers):**

Phase 5.4 bumped `plugin.json` and `lz-advisor.plan/SKILL.md` to `0.7.0` but did **not** bump `lz-advisor.execute/SKILL.md`, `lz-advisor.review/SKILL.md`, or `lz-advisor.security-review/SKILL.md` — those three stayed at `0.6.0` despite Phase 5.4 shipping changes affecting all four skills (D-11 allowed-tools ladder, Common Contract rule 5/5a/6, Mechanism C narrative isolation).

Phase 5.5 CONTEXT D-12 chose to resolve this drift in a single convergent step: all five files advance directly to `0.8.0`. This is not a re-opening of D-12 — the target version is `0.8.0` for all five files per CONTEXT.md. The two-minor-version jump for execute/review/security-review is intentional and documented.

Future plans should treat `0.8.0` as the new uniform baseline. Any new drift (files diverging from the canonical plugin.json version) should be flagged in the next phase's RESEARCH.md drift detection section.
