## Sampling Rate

- **After every task commit:** Run the quick smoke test (`DEF-response-structure.sh`).
- **After every plan wave:** Run the quick smoke test -- plus grep-based cue reachability checks on `context-packaging.md` and both SKILL.md orient phases.
- **Before `/gsd-verify-work`:** Full Stage 2 UAT replay must pass the D-11 per-skill gates (S1/S4 <= 1 advisor call, S5 <= 2, S3/S6 <= 1) and the D-05 word-count gate (S1/S4 Strategic Direction <= 100 words post-`**Critical:**`-strip).
- **Max feedback latency:** ~500 seconds (quick smoke test end-to-end including 3 `claude -p` calls).
