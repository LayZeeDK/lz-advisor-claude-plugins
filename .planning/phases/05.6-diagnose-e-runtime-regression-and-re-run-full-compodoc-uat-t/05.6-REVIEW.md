### IN-01: Output Constraint narrative still phrases the frame requirement in the singular

**File:** `plugins/lz-advisor/agents/advisor.md:58`
**Issue:** The Output Constraint paragraph that governs the unverified-context
frame says `the numbered item MUST begin with the literal sentence frame
...` -- singular "the numbered item". With density bifurcation in place, the
new thin-context example block legitimately opens with the frame on four
consecutive items. The prose is still technically correct (each qualifying
item MUST begin with the frame, so four qualifying items yield four frames),
and the Edge Cases section later speaks in terms of "the numbered item that
depends on the unverified context" which is compatible. This is not a
contract break, but the narrative can be tightened so nothing in the
governing paragraph suggests a one-item-at-most reading -- especially now
that a worked thin-context example demonstrates four framed items in a row.
**Fix:** Optional polish in a future version. Consider a minor rewording of
the first sentence of that paragraph, for example:

```markdown
When your advice depends on context the executor did not package (framework,
runtime, environment, auth model, transport, caller behavior, or any fact not
stated in the prompt), every such numbered item MUST begin with the literal
sentence frame `Assuming X (unverified), do Y. Verify X before acting.` This
is an unconditional output contract, not an edge case. See Edge Cases for
the full substitution rules, worked examples (both full-context and
thin-context), and the forbidden-paraphrase list.
```

Severity justification: Info (not Warning) because (a) the Edge Cases block
on lines 153-177 already anticipates multi-frame responses explicitly ("If
the task is so thin-context that every numbered item would need the frame,
state the two or three most load-bearing assumptions inline..."), and
(b) the thin-context density example itself is canonical at the agent level
-- the Opus advisor will read both the narrative and the example and
reconcile them from the example.
