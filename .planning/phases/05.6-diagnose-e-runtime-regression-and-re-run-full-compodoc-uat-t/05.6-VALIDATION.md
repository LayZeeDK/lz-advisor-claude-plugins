## Sampling Rate

- **After every task commit:** `rg -n "<anchor string>"` for the specific edit surface (e.g., `rg -c "^### Density example \(thin-context" plugins/lz-advisor/agents/advisor.md` after Plan 02 task)
- **After every plan wave:** `bash .planning/phases/05.4-address-uat-findings-a-k/smoke-tests/DEF-response-structure.sh` (Stage 0 gate after Wave 2 merges)
- **Before `/gsd-verify-work`:** Stage 2 UAT replay (6 sessions) + fresh tally + D-11 per-session gate pass OR documented human checkpoint signal
- **Max feedback latency:** Quick run ~180 seconds (Stage 0 smoke); Full-suite ~25 minutes (Stage 2 UAT replay)
