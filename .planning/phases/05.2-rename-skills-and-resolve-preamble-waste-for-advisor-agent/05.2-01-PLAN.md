<objective>
Rename all four skill directories and eval workspace directories using git mv, update all
skill frontmatter (name, version, description trigger phrases), update skill body text
sibling references from colon-form to dotted-form, change advisor agent effort from high
to medium, and bump plugin.json version to 0.3.0.

Purpose: Skill names currently clash with other plugins (bare `plan`, `execute`, etc.).
Dotted names (`lz-advisor.plan`) give unique user-facing shorthands. Advisor effort: high
causes Opus to explain its plan before acting, wasting the single available turn on
preamble narration. Effort: medium eliminates this per Anthropic docs.

Output: Renamed skill directories with updated frontmatter, advisor agent with effort:
medium, plugin.json at 0.3.0, eval workspace directories renamed.
</objective>
